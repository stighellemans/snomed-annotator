# Agentic SNOMED linker

This folder contains the Python CLI tools used by the repository:

- `snomed_vector_db.py`: builds a local SNOMED vector DB or FAISS index from an RF2 release path.
- `semantic_link_snomed.py`: vector-only SNOMED linker.

See the root [`README.md`](../README.md) for setup, vector DB build commands, and recommended run modes.
